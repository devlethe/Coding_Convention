# Coding Convention

해당 문서는 개인적으로 사용하는 코딩 컨벤션에 대해 기록한 문서입니다.<br>

* 기본적으로 코딩 스타일은 K&R을 사용합니다.
* 하지만, 팀원의 스타일에 따라 BSD도 사용합니다.

```JAVA
// K&R
if(...) {
    doSomething();
}

// BSD
if(...)
{
    doSomething();
}

```
